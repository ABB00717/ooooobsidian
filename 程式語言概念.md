# 怎麼會需要這個學科？
語言都有規則（語言遊戲）。
日常語言 -> 人腦自己翻譯 -> 情感、反應
程式語言 -> 翻譯器把程式翻譯成一條條指令 -> 機器依照指令運行

程式語言如果不夠接近日常語言，人類就很難寫；但若沒有嚴格規範，又不可能翻譯成指令。這裡就是在學有哪些規範。

現代電腦流行馮紐曼架構，有運算單元、儲存單元。讀取指令後，根據指令操作，改變狀態。而圍繞記憶體，我們需要知道如何解讀內部的值、以及怎麼自訂結構、命名相關的規則。對於指令讀取，還有流程控制以及抽象化。最後抽象化的方法就決定語言的風格。
